---

## macOS / Linux

```bash
curl -fsSL https://install.seasnoke.dev | bash
```

## Windows PowerShell

```powershell
irm https://install.seasnoke.dev/windows | iex
```

## Homebrew

```bash
brew tap seasnoke/tap
brew install seasnoke
```
